University Carpooling & Ride Sharing App

How to Run the Application
  
  Prerequisites
  
    Node.js (v16 or later)
    
    npm or yarn
    
    Expo CLI
    
    Expo Go app (mobile device)
  
  Install Expo CLI 
    npm install -g expo-cli
  
  Clone the Repository
    (In bash)
    git clone https://github.com/Mohamedeid710/university-carpooling-app
    cd university-carpooling-app
  
  Install Dependencies
      (In bash)
      npm install
      or
      yarn install
  
  Start the Application
      (In bash)
      expo start
      Run
  
  Scan the QR code using the Expo Go app
  Or run on an Android emulator / iOS simulator
